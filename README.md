# Northwest-digital1
NorthWest Digital – Portfolio Website
Overview
This project is the official portfolio website for NorthWest Digital, a small digital service brand focused on helping businesses in Bamenda, Cameroon improve their online presence.
The goal of this website is to demonstrate the type of professional websites that can be built for local businesses such as hotels, restaurants, hostels, and service providers.
The site showcases services, a sample portfolio project, and provides a way for potential clients to contact NorthWest Digital.
